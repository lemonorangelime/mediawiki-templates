# Template:Flex columns

Generates a column flex-box

## Usage example: 

{{Flex columns
|1 =
<div>1</div>
<div>2</div>
|2 = 
<div>3</div>
<div>4</div>
}}

### Creates this table:

---------------------------------------------------------------------------
|1                                  2                                     |
|3                                  4                                     |
---------------------------------------------------------------------------
